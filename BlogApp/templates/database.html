Database in Python

Databases storage implementations vary in complexity. SQLite, a database included with Python, creates a single file for all data per database.Other databases are such as PostgreSQL, MySQL, Oracle and Microsoft SQL Server etc.


Most Common Database

PostgreSQL and MySQL are two of the most common open source databases for storing Python web applications' data.

MySQL Database

MySQL is an interface for connecting to a MySQL database server from Python. It implements the Python Database API v2.0.


How do I install Mysql?

For Python command prompt, use the following command -

pip install mysql.connector

Syntax and example for connecting database
import mysql.connector

# Open database connection
db=mysql.connector.connect(host="hostname",user="username",password="password",database="databasename")
db = mysql.connector.connect(host="localhost",user="testuser",password="test123",databse="test" )

# prepare a cursor object using cursor() method
cursor = db.cursor()

# execute SQL query using execute() method.
cursor.execute("SELECT VERSION()")

# Fetch a single row using fetchone() method.
data = cursor.fetchone()
print "Database version : %s " % data

# disconnect from server
db.close()
